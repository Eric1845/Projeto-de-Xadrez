Um site para o clube

Esse é o site do clube de xadrez do IFMO! O propósito desse site é de administrar o clube e indicar aulas para que todos possam aprender xadrez.

Aqui estarão disponíveis videoaulas, livros e muitos outros materias para que todos possam apender e jogar xadrez. Também será possível fazer o cadastro no clube, para que aqueles que têm interesse participem das atividades e competições.

Sobre o site

O site conta com a página inicial (esta), uma página para cadastro e uma página para aulas de xadrez. Também tem acesso a um banco de dados conectado ao servidor, que registrará os dados dos alunos inscritos no clube.
